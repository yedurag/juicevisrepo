This is a simple demo web application to explore the products and ingredients for main juice brands like the Juicy Juice. Right now the app is limited to three juice brands.

Head over to the products or ingredients page to start exploring. The process book is about how I developed this application in three days.

Powered by

This web application is powered by-

Nutritionix API the data source,

Flask the python web framework,

Dc.js the dimensional charting javascript library,

Crossfilter.js the multidimensional filtering javascript library,

D3.js the data driven document manipulating javascript library,

Bootstrap-select the jquery plugin for dropdowns,

Bootstrap the html, css and js framework for responsive layouts,

and

Jquery the feature rich javascript library
